# 연차나우 (LeaveNow) - 2026년 황금연휴 가이드 ✈️

2026년 대한민국 공휴일과 대체공휴일을 분석하여, 직장인들에게 최적의 연차 사용 타이밍을 제안하는 스마트 플래너입니다. 단 1~2일의 연차로 즐기는 최장의 휴가 스케줄을 확인해보세요.

## 🚀 주요 기능

- **2026년 황금연휴 TOP 3**: 가성비가 가장 높은(휴식일/연차사용일 비율) 연휴를 한눈에 보여줍니다.
- **맞춤형 일정 추천**: 사용 가능한 연차 개수를 입력하면, 1년 중 가장 효율적인 연차 사용 계획을 자동으로 설계합니다.
- **인터랙티브 연간 달력**: 12개월 전체 달력에서 공휴일과 연차 권장일을 시각적으로 확인할 수 있습니다.
- **이미지로 저장 (Viral)**: 나만의 연차 계획을 고화질 이미지로 저장하여 팀원이나 친구들과 공유할 수 있습니다. (Confetti 효과 포함!)
- **콘텐츠 언록 (수익화)**: 스폰서 링크 확인을 통해 맞춤 추천 기능을 활성화하는 비즈니스 모델이 통합되어 있습니다.

## 🛠 기술 스택

- **Frontend**: React (Vite), Tailwind CSS v4, Framer Motion
- **Icons**: Lucide React
- **Animations**: Lottie React, Canvas Confetti
- **Utils**: html2canvas (이미지 캡처)
- **Deployment**: Firebase Hosting
- **Analytics**: Firebase Analytics

## 📦 설치 및 실행

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 로컬 실행:
   ```bash
   npm run dev
   ```

3. 빌드 및 배포:
   ```bash
   npm run deploy
   ```
